Capítulo: A Minotaura Fêmea

Minotauros são homens. Não importa qual mulher de qual raça gere cria a um filho dos homens-touro. Se nascer com chifres, será um homem.
A menos assim se pensava há trinta anos. Uma mulher abusada por um monstro dá a luz a uma pequena mocinha. Ela foi batizada com o mesmo nome do lugar que estaria destinada a trabalhar pelo resto da sua curta vida. Mina, a Minotaura.
